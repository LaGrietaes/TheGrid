# Media Ingest Placeholder Assets

Drop user-provided placeholder images in this folder so Media Ingest cards can render by file type when no thumbnail is available.

Supported keys and preferred filenames:
- Raw: `raw.png`
- Video: `video.png` (or `mp4.png`)
- Photo: `photo.png` (or `image.png`)
- Audio: `audio.png` (or `sound.png`)
- PSD: `psd.png`
- AI: `ai.png`
- Document: `doc.png` (fallback)
- Drone: `drone.png` (or `uav.png`)

Notes:
- PNG is recommended.
- Suggested size: 768x768 or 1024x1024.
- Keep transparent or dark background for best blend with the UI theme.
